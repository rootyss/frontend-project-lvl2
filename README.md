## Setup

```sh
$ git clone https://github.com/rootyss/frontend-project-lvl2
$ make install (in cloned folder)
```
for help:
```sh
$ gendiff -h or gendiff --help
```

## Usage
```sh
$ gendiff [path1] [path2]
```
